Tags¶

Tags are a very important part of SARF as they provide metadata to connect multiple tools.

You can create your own tags but if there is a tag registered for your purpose you should use it.

If you create a custom tag for a tool please make a Merge Request adding it to this file.

Registered tags¶

  • tool:nmap

  • format:nmap_xml

  • tool:nikto

  • format:nikto_json

  • tool:zap

  • tool:burp

  • format:png
